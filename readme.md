# java 12 to java 17 new features example

## Develop environment
1. java 17.0.1 with sdkman manage
2. gradle 7.3 with sdkman manage
3. idea2021 community
4. macos or windows

## How to run

1. execute command
```
./gradlew idea
```
2. double click java-feature.ipr

3. execute command
```
./gradlew test
```
